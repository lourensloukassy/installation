LOKU REPO INSTALASI

Buka Kodi Media Center

Kemudian Pilih Setting, dan pilih File Manager

Pilih Add Source, dan masukkan link "https://lourensloukassy.github.io/iptv/" ,

kemudian save dengan nama "LOKU REPO"

Kemudian pergi ke System, Pilih Advanced

Pilih Addons, Lalu enable "Unknown Sources" dan "Show Notifications"

Update Official Addons From pilih "Any Repositories"

Kembali ke setting, lalu pilih Addons

Pilih "install from zip file"

Pilih "LOKU REPO"

Tunggu sampai "LOKU REPO" memasuki zip file bernama

"repository.loku-1.0.0.zip" , kemudian klik dan proses instalasinya otomatis

Apabila sudah selesai dan berhasil akan muncul notifikasi

"Loku Repository Add-on Installed"

Masih di addons, "pilih install from repository"

Pilih Loku Repository , dan akan muncul tampilan berikut

Install Addons Loku Repo , tunggu sampai selesai

Install Addons TheOath , lakukan seperti biasa

Install SMR Link Tester , lakukan seperti biasa

Kembali ke Loku Repository pilih "program-addons",

Install Orion

Apabila sudah melakukan instalasi,

Kembali ke tampilan awal atau home, pilih Addons

Loku Repo, TheOath sudah terinstall
