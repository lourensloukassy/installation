Instalasi Ultrasurf

Apabila sudah mendowload aplikasi ultrasurf, ikutin langkah langkah berikut ini :

* Microsoft Windows : buat sebuah folder baru dengan nama "Ultrasurf" pada path drive "C:\" , kemudian pindahkan file aplikasi ultrasurf yang sudah di download ke "C:\Ultrasurf\" , lalu click kanan pada ultrasurf aplikasinya pilih send to > desktop shortcuts

Result : "C:\Ultrasurf\usf.exe"

It's recommended.

* macOS Apple : extract file zip cloudfare, kemudian buka instalasi dan klik continue ; lakukan instalasi seperti biasa hingga selesai
