LOKU REPO INSTALASI


PVR IPTV Simple Client (Regional Indonesian)

Buka Kodi Media Center

Kemudian pilih setting, lalu pilih Addons

Masih di addons, "pilih install from repository"

Pilih Kodi Add-On Repository , dan akan muncul tampilan berikut

Cari PVR Clients , lalu masuk dan tampilannya sebegai berikut

Install PVR IPTV Simple Clients dan lalu pilih konfigurasi

Setting konfigurasi base url "M3U URL Playlist" pada tab "General"

Masukkin Link Berikut -> "https://bit.ly/3g5OCQ4"

Apabila sudah memasukkan link tersebut, aktifkan timeshift kemudian klik "OK"

Kembali Ke Home, dan pilih Exit

Mulai dari awal lagi, Buka Kodi Media Center kemudian pilih "TV"

Tunggu sampai loading PVR Manager completed 100%

Apbila sudah sukses , akan muncul gambar sebagai berikut

Noted : Pastikan saat anda memulai memutar TV Nasional , tidak menggunakan VPN
